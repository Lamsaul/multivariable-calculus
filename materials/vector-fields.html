Analyzing Vector Fields

Divergence

  • A measurement of how much fluid/flow enters the neighborhood around \(P\) compared to how much leaves.
  • If more fluid enters than leaves, then the divergence will be negative at \(P\).
  • If the same amount of fluid enters and leaves, then the divergence will be \(0\) at \(P\).
  • For

    \[ F(x,y,z)=P\hat{i}+Q\hat{j}+R\hat{k} \]

    \[ Div(F) = \nabla \cdot F = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} \]

  • It is a scalar quantity.

Curl

  • A measurement of the rotation of the vector field in the neighborhood around \(P\).
  • If curl is \(+\) at a point, then the flow would rotate counterclockwise.
  • And if \(-\), then clockwise.
  • If \(0\), then there is no rotation at \(P\).

  • \[ Curl(F) = \nabla \times F \]

  • It is a vector.

Work Done to Move along a Vector Field

Non-Conservative Vector Fields

  • For a non-conservative vector field \(F(x,y,z)=F(x(t), y(t), z(t))\) along a path \(r(t)\), the work done for a particle to move along the given path \(C\) is given by,

    \[ w=\int_C F \cdot dr = \int_{t=a}^{t=b} F(r(t)) \cdot r'(t) \cdot dt \]

  • These integrals are *path dependent*.

Conservative Vector Fields

  • \(F(x,y)\) is conservative iff \(F(x,y)=\nabla f(x,y)\) for some \(f(x,y)\) that is called a potential function.
  • This means a line integral within a conservative vector field can be expressed by,

    \[ \int_C F \cdot dr = \int_C \nabla f \cdot dr = f(b)-f(a) \]

Showing \(F\) is Conservative

  • For a conservative \(F(x,y)=P\hat{i}+Q\hat{j}\),

    \[ \frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x} \]

  • Or for \(F(x,y,z)=P\hat{i}+Q\hat{j}+R\hat{k}\),

    \[ Curl(F)=0 \]
