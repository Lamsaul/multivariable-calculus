What is a Vector Function?

  • A vector function is a parametrically defined function where the terminal points of vectors trace a curve instead of a surface in 3 dimensional space.
  • The fact that \(t\) has a certain domain gives \(r(t)\) an orientation.

Sketching a Vector Function

  • Identify \(x, y, z\).
  • Use one or more components to get a curve or surface (get rid of \(t\)).
  • For just two components, sketch on a plane.
  • For three components, it's a curve travelling along a surface.
  • Use values of \(t\) to find points and the orientation.

Derivative of a Vector function

  • The derivative of a vector function describes the *direction vector of the tangent vector to the curve at some value of \(t\).
  • The derivative of a function is defined as,
  • The derivative of a function is defined as,
    • \[f'(x) = \lim_{\Delta x \to 0} \frac{f(x+\Delta x) - f(x)}{\Delta x}\]
  • Similarly, the derivative of a vector function is defined for some parameter \(t\) as,
    • \[r'(t) = \lim_{\Delta t \to 0} \frac{r(t+\Delta t) - r(t)}{\Delta t} = \lim_{\Delta t \to 0} \frac{f(t+\Delta t) \hat{i} + g(t+\Delta t) \hat{j} + h(t+\Delta t) \hat{k}-(f(t) \hat{i} + g(t) \hat{j} + h(t) \hat{k})}{\Delta t}\] \[=\lim_{\Delta t \to 0} \frac{f(t+\Delta t)-f(t)}{\Delta t} \hat{i} + \frac{g(t+\Delta t)-g(t)}{\Delta t} \hat{j} + \frac{h(t+\Delta t)-h(t)}{\Delta t} \hat{k} = f'(t)\hat{i} + g'(t) \hat{j} + h'(t) \hat{k}\]
  • Example: The derivative of \(r(t) = t \hat{i} + t^2 \hat{j} + t^3 \hat{k}\) is \(r'(t) = \hat{i} + 2t\hat{j} + 3t^2 \hat{k}\).

The Arc Length of a Curve

  • For some vector function \(r(t)\) with a derivative \(r'(t)\), the so-called 'magnitude' of \(r'(t)\) is expressed as,
    • \[||r'(t)|| = \sqrt{(f'(t))^2 + (g'(t))^2 + (h'(t))^2}\]
  • The arc length of the curve generated by the vector function, for some interval \(t \in [a,b]\), is given as,
    • \[L=\int_a^b \sqrt{(f'(t))^2 + (g'(t))^2 + (h'(t))^2} \cdot dt = \int_a^b ||r'(t)|| \cdot dt\]

Arc Length Reparametrization

Why?

  • We can 'walk through the curve' instead of using a vector to point at each individual point.

How?

  • Find the arc length function (\(S(t)\) not \(L\)).
    • \[S(t)=\int_a^t ||r(u)|| \cdot du\]
  • \(a\) is where \(t\) begins.

Example

    \[r(t) = e^t \cos{t} \hat{i} + e^t \sin{t} \hat{j} + e^t \hat{k} \text{ for } t \geq 0\] \[||r'(t)|| = \sqrt{3}e^t\] \[S(t) = \int_0^t \sqrt{3} e^u \cdot du = \sqrt{3}(e^t-1)\] Turn \(S(t)\) into the parameter \(S\), \[S=\sqrt{3}(e^t-1)\] \[e^t = \frac{\sqrt{3}}{3} S + 1\] \[t=\ln{\left(\frac{\sqrt{3}}{3} S +1\right)}\] Thus, \[r(s) = \left(\frac{\sqrt{3}}{3}S+1\right)\cos{\left(\ln{\left(\frac{\sqrt{3}}{3} S +1\right)}\right)}\hat{i} + \left(\frac{\sqrt{3}}{3}S+1\right)\sin{\ln{\left(\frac{\sqrt{3}}{3} S +1\right)}}\hat{j}+\left(\frac{\sqrt{3}}{3}S+1\right)\hat{k}\]
