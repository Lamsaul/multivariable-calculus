Problem 2

Let S be the part of the spherical surface \(x^2 + y^2 + z^2 = 4\), lying in \(x^2 + y^2 < 1\), which is to say outside the cylinder of radius one with axis the z-axis.
  1. Compute the flux outward through \(S\) of the vector field \(\vec{F} = y\hat{i} -x\hat{j}+z\hat{k}\)
  2. Show that the flux of this vector field through any part of the cylindrical surface is zero.
  3. Using the divergence theorem applied to \(\vec{F}\), compute the volume of the region between \(S\) and the cylinder.
    4. Solution

Problem 3

  1. Parametrize \(C\) and use the parametrization to evaluate the line integral
    2. \[I = \oint_C xz\cdot dx + y \cdot dy + y \cdot dz\]

  3. Write down a flux integral through \(S\) which can be computed using the value of \(I\).
    4. Solution
